Other similar software

Free

Useful tools with no funds needed.

  • Flasm - Flasm on SourceForge
    - AS1/2 (dis)assembler
  • RABCDasm - RABCDasm on Github
    - AS3 (dis)assembler, FFDec uses similar AS3 p-code syntax
  • SWF iD - SWF iD homepage
    - Great tool to identify well known obfuscators / SWF signatures
  • Free SWF Decompilers - Post on Bruce Jawns blog
    - Large list of free Flash decompilers
